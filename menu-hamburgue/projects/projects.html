Help Desk

Um projeto de sistema para help desk

Esse sistema está sendo criado com spring boot com a linguagem java com banco de dados postgres e para front angular.
Nele estou aprendendo conceitos de segurança e controle de acesso, assim fazendo que nem todos os usuários possuam o mesmo controle do sistema, criando prioridades de entregas e status do andamento do serviço que está sendo feito.
Com o uso do angular estou aprendendo o uso das chaves de acesso com jwt e como utilizar o Angular Material para facilitar no design das páginas e criando elas de modo mais dinâmico.
Este sistema tem como intuito principal realizar o encurtamento de URL padrão para uma URL encurtada.
Neste projeto foi desenvolvido com o intuito de aprender mais sobre o spring security e a parte de autenticação de usuários, porém todos os usuários têm o mesmo direito no sistema nenhum possui restrição. Nele foi feito o CRUD apenas de URL, os usuários ja estavam cadastrados no sistema. Nele pude aprender um pouco mais também como realizar a geração de um código aleatório com a dependência do guava disponibilizada pelo google que faz essa geração do código.
Este sistema foi feito com spring boot, java, dependências maven, docker com postgres e angular.
